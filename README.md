
## 🧩 功能模組
- **攻擊系統**：玩家造成傷害，敵人反擊
- **招架小遊戲**：限時按鍵判定，成功閃避傷害
- **天賦系統**：可配置點數強化攻擊、防禦、招架
- **視覺反饋**：血量更新、提示文字、動畫閃光
